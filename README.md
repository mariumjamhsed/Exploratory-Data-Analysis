# Exploratory Data Analysis - Classic Datasets

Comprehensive EDA on three famous datasets using pandas, matplotlib, and seaborn.

## Datasets Analyzed

### 1. Iris
- 150 flower samples, 3 species
- Key Analysis: Species separation by measurements
- Visualizations: histplots, boxplots

### 2. Tips
- 244 restaurant transactions
- Key Analysis: Factors influencing tipping
- Visualizations: Scatter plots,Kde plots

### 3. Titanic
- 891 passengers
- Key Analysis: Survival patterns by class, gender, age
- Visualizations: Scatter plots, histplot, Strip plot, kdeplot, heatmap

## Key Findings

**Iris:** The three species are clearly separable using petal measurements.

**Tips:** Bill amount has the strongest correlation with tip. Weekend and dinner customers tip more.

**Titanic:** Females and 1st class passengers had much higher survival rates.

## Technologies Used
- pandas (data analysis)
- matplotlib (plotting)
- seaborn (statistical visualizations)
